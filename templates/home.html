// Security First

Encrypted in your browser.
Invisible to everyone else.

We can't read your files. Even if we wanted to. Your data is encrypted locally using AES-256 before it ever leaves your device. The server receives nothing but scrambled noise.

Client-Side E2EE

Encryption happens in your browser using the Web Crypto API. Keys never leave your device.

Zero Server Trust

The server only receives and stores encrypted blobs. Database admins cannot access file content.

In-Memory Decryption

Files decrypt in RAM and download natively. Zero decrypted traces remain on the server.
